Instructions

1. Enter your SellMyApp Order#.
2. Specify a level number (1 or greater) followed by a colon. Below each level heading, specify a stack with a dash followed by a list of colors, separated by spaces.
3. Repeat for as many levels as you want in the game. Empty lines are permitted between levels.
4. Click 'Generate' to download a zip file of your levels. Unzip this file and move all of the data files into the SortCandy3D/Assets/Resources folder of the Unity project.

Colors: yellow, pink, violet, brown, blue, green, lime
Empty space: o

Note: This tool does not save the above level text. Instead, since the level format is valid YAML, save the above as 'levels.yaml' on your local machine for backup purposes.
